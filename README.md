# 弾幕キーボード (Danmaku Keyboard)

## 概要

「弾幕キーボード」は、あらかじめ登録しておいた好きな単語を、ボタン一つで素早く入力・送信できるカスタムAndroidキーボードアプリです。
配信アプリのコメント欄などで、同じコメントを素早く何度も送りたい、といった用途に特化しています。

## 主な機能

*   **ワンタップ入力**: 画面に表示される2つのボタンをタップするだけで、設定した単語を簡単に入力できます。
*   **インテリジェント送信**: LINEなどのチャットアプリでは入力と同時にメッセージを送信し、ブラウザでは検索を実行します。アプリの状況を判断して、最適なアクションを自動で実行します。（※一部のカスタムUIを持つアプリでは動作しない場合があります）
*   **かんたん単語設定**: キーボードのボタンを長押しするだけで、単語リストが表示され、ボタンに割り当てる単語をすぐに変更できます。
*   **専用の単語管理画面**: アプリアイコンからアプリを起動すると、単語の追加や削除ができる専用の管理画面が開きます。

## 使い方

1.  **キーボードの有効化**
    *   Androidの `設定` > `システム` > `言語と入力` > `画面上のキーボード` を開きます。
    *   `キーボードの管理` をタップし、「弾幕キーボード」をオンに切り替えます。
2.  **キーボードの切り替え**
    *   文字を入力したいアプリ（LINEなど）を開きます。
    *   キーボードが表示されたら、画面の右下などに表示されるキーボードアイコンをタップし、入力方法を「弾幕キーボード」に切り替えます。
3.  **単語の設定**
    *   キーボードに表示されている「+」ボタンを長押しします。
    *   単語の選択画面が開くので、ボタンに設定したい単語をタップします。
    *   （単語が空の場合）スマートフォンのアプリ一覧から「弾幕キーボード」を開き、右下の「+」ボタンから新しい単語を追加してください。
